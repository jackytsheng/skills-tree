# skills-tree

## create virtual environment
```
python3 -m venv --without-pip .venv
```
## Use it 
```
source .venv/bin/activate
```
## create pip
```
python3 -m ensurepip --upgrade
```


## create virtual environment
```
python -m pip install -r requirements.txt
```
